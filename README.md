# 🏨 Hotel Chatbot

This is a Hotel Management Chatbot built using HTML, CSS, JavaScript and Python Flask.

## 🚀 Features
- Room information
- Price details
- Booking support
- Facilities like WiFi, parking, etc.

## ▶️ How to Run
1. Run backend:
   python chatpython_flask.py

2. Open index.html
